# Site_web_S4

## Objectif :

Créer une banque d'image corporate visant à aider les entreprises à créer leur identitée. Le site doit permettre de thematiser (free, nature, city, logo, portefolio...) les images (background, logos...), et de les charger progressivement en fonction du scroll de l'utilisateur. On doit pouvoir entrer en communication avec les designers (simple formulaire de mail), si ceux-ci l'ont autorisé. L'utilisateur doit pouvoir créer un compte pour acheter, télécharger ou poster des images. Un système de paiement doit être intégré.

A voir pour la modération des images postées.
